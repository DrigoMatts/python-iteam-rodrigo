# Projeto Integrador - Urna Eletrônica

Este projeto consiste em um sistema de votação eletrônica desenvolvido em Python, utilizando os princípios de Orientação a Objetos (POO) para garantir organização, encapsulamento e escalabilidade.

## Funcionalidades
- **Cadastro de Candidatos:** Gerenciamento dos dados dos candidatos através da classe `Candidato`.
- **Processo de Votação:** Sistema centralizado na classe `Urna` que valida os votos.
- **Tratamento de Exceções:** Implementação de exceções customizadas para erros de entrada e candidatos inexistentes.
- **Relatórios:** Exibição clara dos resultados após o encerramento da votação.

## Como rodar o projeto
1. Certifique-se de ter o Python instalado.
2. No terminal, navegue até a pasta `projeto-integrador`.
3. Execute o comando: `python urna_eletronica.py`
4. Siga as instruções no terminal: digite o número do candidato ou 'fim' para encerrar.

## Tecnologias utilizadas
- Python 3
- Orientação a Objetos (Classes, Properties, Encapsulamento)
- Tratamento de Exceções customizadas

O enunciado completo está no arquivo `projeto-integrador-urna.pdf` disponibilizado pelo professor.

Coloque seus arquivos `.py` nesta pasta. Recomendado:

- `candidatos.py` — classes de candidatos (se usar módulos separados)
