### R/Explicação:
lista_a = [1, 2, 3]
lista_b = [1, 2, 3]

print(lista_a == lista_b) # True: Os valores dentro são iguais.
print(lista_a is lista_b) # False: São dois objetos diferentes na memória.

Quando usar is em código,is para comparar valores (como números ou strings) ele é para comparar com o objeto None ou com objetos singletons.
Porque None é um objeto único (singleton) na linguagem Python. Usar is é mais rápido e semanticamente correto para verificar se uma variável foi inicializada ou se uma função não retornou nada.
