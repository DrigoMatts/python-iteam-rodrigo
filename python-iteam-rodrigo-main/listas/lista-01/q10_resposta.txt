### Indentificação de problemas: 
O uso de alunos, Listas são mutáveis, e esse valor padrão é criado apenas uma vez, na definição da função. 
Se você chama a função múltiplas vezes sem passar argumentos, todas as chamadas compartilharão a mesma lista modificada o correto seria é usar None ao inves de ficar sempre chamando a função alunos.
Usar for i in range para acessar elementos via índice (alunos[i]) ao invés de usar diretamente sobre os elementos da lista (for aluno in alunos)
Na expressão aprovados = aprovados, ele cria uma nova lista a cada iteração, copiando todos os elementos antigos e adicionando o novo seria melhor usar o .append()
para adicionar os itens ao final de uma lista criada.
