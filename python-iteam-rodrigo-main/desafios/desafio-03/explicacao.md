## Resposta à Pergunta Obrigatória

> Por que usamos `elif` e não múltiplos `if` separados? Dê um exemplo concreto onde a diferença causaria um resultado errado.

_(Sua resposta aqui — use suas próprias palavras.)_

### R: Usamos o "Elif ou Else" para criar uma cadeia de exclusão. Quando o python encontra uma condição verdadeira em um bloco, ele executa o código correspondente e ignora todo o resto daquela estrutura. Se você usar múltiplos if separados, o Python testará todas as condições de forma independente, mesmo que a primeira já tenha sido satisfeita.
